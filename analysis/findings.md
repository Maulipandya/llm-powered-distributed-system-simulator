# Findings

## Summary
The simulator compares static and hybrid (LLM-assisted) modes across multiple cluster sizes.

## Main Observations
- Hybrid mode improves throughput
- Hybrid mode introduces additional latency overhead
- Recovery behavior improves with scale
- Results show a trade-off between adaptability and cost

## Conclusion
The experiments support the paper’s claim that LLM-assisted simulation improves flexibility while introducing measurable overhead.
